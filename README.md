## Project setup
Ketika pertama kali mendownload repository ini, masuk ke folder proyek lalu jalankan perintah berikut:
```
npm install
```

## Install library
Instal vue
```
npm install vue
```
Kalau gak berhasil, coba:
```
npm install -g @vue/cli
```
Instal Web3
```
npm install web3
```

## Menjalankan program
```
npm run serve
```
